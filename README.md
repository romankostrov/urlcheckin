🚀 LinkChecker Desktop

Профессиональный десктоп-инструмент для массовой проверки ссылок, изображений и медиафайлов.

Разрабатывался как практическое решение для задач SEO, маркетинга и технического аудита сайтов.

🔥 Возможности

✔ Массовая проверка URL (HTTP статус, редиректы)
✔ Проверка изображений на целостность
✔ Базовая проверка видео и файлов
✔ Поддержка CSV / Excel (XLSX / XLS)
✔ Выбор нужного столбца со ссылками
✔ Многопоточная обработка (быстро даже на больших объёмах)
✔ Экспорт результата в Excel
✔ Удобный современный интерфейс (PySide6)

🧠 Что именно проверяет

Программа делает не просто “200 OK”, а полноценную проверку:

доступность ссылки
финальный URL (редиректы)
тип ресурса (страница / изображение / видео / файл)
content-type
размер файла
валидность изображения (битое / не битое)
базовая проверка видео (по сигнатуре)
комментарий по каждому результату
📦 Пример результата
URL	Статус	Тип	Валиден	Комментарий
site.com/img.jpg	200	image	YES	JPEG OK
site.com/video.mp4	200	video	YES	MP4 detected
site.com/broken.png	200	image	NO	Broken image
🖥 Интерфейс
минималистичный стиль (в духе Apple UI)
удобный выбор файла
переключатели режимов проверки
быстрый запуск проверки
готовность к масштабной работе
⚙️ Установка
1. Установить зависимости
pip install -r requirements.txt
2. Запуск
python link_checker_desktop_pyside6_fixed.py
🏗 Сборка .exe
py -m PyInstaller --onefile --windowed --name LinkCheckerDesktop link_checker_desktop_pyside6_fixed.py

Готовый файл:

dist/LinkCheckerDesktop.exe
📊 Когда это реально полезно

Этот инструмент закрывает задачи:

SEO-аудит сайтов
проверка битых изображений
контроль выгрузок товаров (Avito / CMS / каталоги)
проверка ссылочных баз
контроль качества контента
массовая валидация медиа
⚠️ Ограничения
видео проверяется без глубокого анализа (без ffmpeg)
скорость зависит от количества потоков и сети
некоторые сайты могут блокировать запросы
💡 Планы развития
фильтр “битые файлы” прямо в интерфейсе
отображение результатов в таблице внутри приложения
drag & drop файлов
продвинутая проверка видео через ffprobe
API версия
